Minimal Node.js Development Server with Hot Reload and Auto Restart

This project is a lightweight development server built from scratch using Node.js and WebSockets. It provides automatic browser reload for frontend changes and automatic server restart for backend changes without relying on heavy frameworks or tools.

The goal of the project is to understand how modern development servers work internally by implementing the core features manually.

Key Features

Simple HTTP Server

  • Serves HTML files directly to the browser
  • Handles static files such as JavaScript and CSS
  • Basic content-type handling

Static File Serving

  • Supports multiple frontend assets
  • Serves files from a designated frontend folder
  • Clean Separation of Concerns
